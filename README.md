AICharya
Project Overview
AICharya is an educational website designed to teach basic web development and artificial intelligence concepts to beginners. Developed during the ShlokaDecode 2.0 Hackathon by Team Ctrl + New, the platform offers interactive tutorials, hands-on coding exercises, and AI-powered learning assistance to make foundational programming and AI accessible, engaging, and beginner-friendly.
The name AICharya reflects the fusion of AI (Artificial Intelligence) and Acharya (Sanskrit for "teacher" or "guru"), symbolizing an intelligent mentor guiding learners through their journey in technology.

Description
AICharya is a beginner-oriented learning platform that simplifies the entry into web development and artificial intelligence. It provides structured, bite-sized modules covering:

Web Development Fundamentals: HTML, CSS, JavaScript, responsive design, and basic frontend frameworks.
Introduction to AI: Core concepts such as machine learning, neural networks, natural language processing, and ethical AI â€” explained without heavy mathematics.
Interactive Learning: Live code editors, quizzes, AI-generated hints, and personalized progress tracking.
AI-Powered Assistance: A built-in chatbot that answers questions, debugs code, and explains concepts in simple language.

The platform emphasizes learn-by-doing, enabling users to build real projects (e.g., personal websites, chatbots, or simple predictive models) while receiving real-time feedback.
